# HafidzWeb - Student Management System

## Short Description for Portfolio

**English Version:**
A comprehensive web-based Student Management System built with native PHP and MySQL during college coursework. Features full CRUD operations for managing student records including name, registration number, birth date, gender, and description. Implements form validation, database security practices, and responsive design. Hosted on InfinityFree with live demo functionality.

**Indonesian Version:**
Sistem Manajemen Siswa berbasis web yang dibangun dengan PHP native dan MySQL selama perkuliahan. Menampilkan operasi CRUD lengkap untuk mengelola data siswa termasuk nama, NRP, tanggal lahir, jenis kelamin, dan deskripsi. Mengimplementasikan validasi form, praktik keamanan database, dan desain responsif. Di-hosting di InfinityFree dengan fungsi demo langsung.
